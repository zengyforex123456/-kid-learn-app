# PRD — 幼儿双语趣味识字 APP (MVP)

## 产品概要
3-6 岁幼儿汉字+英文游戏学习 APP，手机/平板通用，纯游戏化零压力启蒙。

## 需求清单

| ID | 需求 | 验收标准 |
|----|------|---------|
| R1 | 游戏闯关主页 | 卡通森林主题，大图标入口，点游戏卡片进入，语音引导 |
| R2 | 森林寻宝游戏 | 场景藏3-5张卡片，点击发音+动画释义，收集解锁贴纸 |
| R3 | 泡泡大作战 | 屏幕飘泡泡，语音报词，点击正确泡泡→奖励特效；错→温柔消散 |
| R4 | 配对大闯关 | 三阶：图文配对→听音选图→听音选字/单词，难度递增干扰项 |
| R5 | 双语词汇库 | 50个词(水果/动物/五官/日常)，中文+英文，3-4岁只认不写 |
| R6 | 贴纸奖励系统 | 完成游戏获贴纸，可查看贴纸背包，收集进度展示 |
| R7 | 语音系统 | 中文温柔童声普通话，英文纯正美音，答对欢呼+礼花，答错温柔提示 |
| R8 | 护眼休息 | 每15分钟强制休息动画30s，降低蓝光，柔光背景 |
| R9 | 家长管控 | 密码入口：设置每日时长、护眼开关、查看学习进度(识字量+游戏次数) |
| R10 | 离线可用 | 首次下载资源包后，所有游戏和音频离线工作 |
| R11 | 自适应布局 | 手机紧凑/平板放大，横竖屏适配，大按钮防误触 |
| R12 | 后端内容管理 | CMS 管理词库/音频/图片，学习数据云同步，家长报告 |
| R13 | 首次引导 | 3页动画引导，教孩子基本操作(点/滑)，跳过后不再显示 |
| R14 | 每日学习提醒 | 本地通知，可设时间，默认关闭 |

## 技术栈

- **前端**: Flutter 3.x + Dart
- **状态管理**: Riverpod 2.x
- **本地存储**: sqflite(结构化数据) + Hive(缓存/配置)
- **音频**: audioplayers + flutter_tts
- **动画**: Lottie(复杂动画) + 内置 AnimatedWidget
- **后端**: Node.js + PostgreSQL + MinIO(音频/图片)
- **离线**: 资源包 zip 下载解压本地加载

## 非功能需求

- 动画帧率 ≥60fps
- 安装包 ≤30MB(不含资源包)
- 冷启动 ≤2s
- 按钮最小 60×60px(幼儿防误触)
- 护眼模式色温 ≤3500K
- 无广告、无付费入口(儿童端)

## UI 方向

- **风格**: 卡通圆润、柔和马卡龙配色
- **组件库**: 全自研游戏组件(CustomPainter + Lottie)
- **字体**: 系统圆体(中文) + 系统无衬线(英文)
- **导航**: 纯图标+语音，无文字菜单
