# /status — 项目完整状态

## 报告内容
1. **当前阶段** — 阶段名称、开始时间、任务描述
2. **PRD** — 需求数量、技术栈
3. **架构决策** — 核心决策列表
4. **修改文件** — 文件数量和列表
5. **已完成任务** — 最近 3 个
6. **下一里程碑** — 预计推进条件

## 操作
读取 `.claude/project-state.md` 和 `.claude/prd.md`（如存在），生成结构化状态报告（≤10 行）。
